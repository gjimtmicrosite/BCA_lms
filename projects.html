💻 4 AI Integrated Practical Projects — BCA with AI

Each project follows a structured 4-phase learning framework: Problem Understanding → Manual Development → Data & Logic → AI Enhancement. Students must complete the manual development phases before accessing AI-assisted tools and automation.

P1: AI Website Analyzer P2: Smart Chatbot P3: Startup Website Clone P4: AI Dashboard

🌐 P1: AI Website Performance Analyzer HTML · CSS · JavaScript Weeks 1–4

Objective: Build a responsive website analyzer that evaluates website speed, SEO basics, responsiveness, and UI quality using AI-generated insights.

AI Tools: ChatGPT · Google Lighthouse · Canva AI · GitHub Copilot

📋 4-Phase Framework

🔵 Phase 1 — Website Research & Planning
  • Study 3 modern websites and identify UI components
  • Learn responsive layout basics
  • Create wireframe on paper
  • Prepare color palette and typography selection
  • Deliverable: Website wireframe + planning document
🟠 Phase 2 — Manual Frontend Development
  • Create website using HTML, CSS, and JavaScript
  • Build navigation, hero section, cards, forms, and footer
  • Implement responsive mobile layout manually
  • Validate all links and UI interactions
  • Deliverable: Functional responsive website
🟢 Phase 3 — Performance & Data Analysis
  • Run Lighthouse performance tests
  • Analyze SEO score and accessibility
  • Create performance comparison table
  • Write Before AI analysis note
  • Deliverable: Website audit report
🤖 Phase 4 — AI Enhancement Lab
  • Use ChatGPT to improve SEO tags and UI text
  • Generate AI-based UI suggestions
  • Use Canva AI for graphics and banners
  • Generate AI accessibility recommendations
  • Create final presentation with AI comparison
  • Deliverable: Enhanced website + AI reflection report + Viva

🤖 P2: Smart AI Chatbot Development JavaScript · AI APIs Weeks 5–7

Objective: Develop a browser-based AI chatbot capable of answering student or customer queries.

AI Tools: ChatGPT API · Dialogflow · JavaScript · Firebase

📋 4-Phase Framework

🔵 Phase 1 — Chatbot Planning
  • Study chatbot workflows
  • Prepare conversation flow diagram
  • Identify common user queries
  • Create chatbot response structure
🟠 Phase 2 — Core Development
  • Build chatbot UI using HTML/CSS
  • Create message handling system in JavaScript
  • Implement predefined responses
  • Test multiple conversation flows
🟢 Phase 3 — Data & Testing
  • Test chatbot accuracy
  • Record response timing
  • Prepare chatbot training questions
  • Write testing summary report
🤖 Phase 4 — AI Integration
  • Integrate ChatGPT API
  • Add AI-generated smart replies
  • Implement sentiment-based responses
  • Improve conversational flow using prompts
  • Deliverable: AI chatbot + deployment + viva

🚀 P3: Startup Website Clone with AI Features React · UI/UX Weeks 8–10

Objective: Recreate and enhance a startup/company website with AI-powered features.

AI Tools: ChatGPT · Figma AI · GitHub Copilot · Canva AI

📋 4-Phase Framework

🔵 Phase 1 — Website Analysis
  • Select startup website
  • Analyze UI components and layout
  • Create page structure and flow
  • Prepare feature comparison sheet
🟠 Phase 2 — Frontend Clone Development
  • Develop responsive clone layout
  • Create reusable UI components
  • Add animations and transitions
  • Build forms and navigation systems
🟢 Phase 3 — Feature Enhancement
  • Optimize responsiveness
  • Improve loading performance
  • Add dark mode or advanced UI features
  • Create comparison report
🤖 Phase 4 — AI Powered Improvements
  • Generate AI UI suggestions
  • Create AI-generated graphics
  • Use Copilot for component optimization
  • Implement AI text generation
  • Deliverable: Enhanced startup website clone + presentation

📊 P4: AI Analytics Dashboard Power BI · JavaScript Weeks 11–12

Objective: Build a modern analytics dashboard with AI-generated insights and data visualization.

AI Tools: Power BI · Excel AI · ChatGPT · Google Looker Studio

📋 4-Phase Framework

🔵 Phase 1 — Dashboard Planning
  • Select dashboard domain
  • Identify KPIs and charts
  • Create layout wireframe
  • Prepare dataset structure
🟠 Phase 2 — Data Handling
  • Clean and organize dataset
  • Create pivot tables
  • Perform manual calculations
  • Validate charts and values
🟢 Phase 3 — Dashboard Creation
  • Create KPI cards and visualizations
  • Build charts and filters
  • Enable responsive dashboard view
  • Prepare dashboard summary
🤖 Phase 4 — AI Insights Integration
  • Generate AI-based analytics insights
  • Add smart recommendation engine
  • Create natural language dashboard summary
  • Implement AI forecasting visuals
  • Deliverable: AI analytics dashboard + viva
